# AWS_SNOWFLAKE_POWERBI-RERORT_ETL_Pipeline
End-to-end cloud data pipeline built using AWS S3, Snowflake (Snowpipe, Streams, Tasks, UDFs), and Power BI. This project automates ingestion of Yelp JSON datasets into Snowflake using Snowpipe, performs real-time transformations with Streams &amp; Tasks, enriches reviews with sentiment analysis, and visualizes insights in Power BI Report Builder.
